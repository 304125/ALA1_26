# Advanced Linear Algebra (ME-ALA1-A26)

Link to course description: https://my.via.dk/media/mitvia/semesteroverblik/kursusbeskrivelser/maskiningenioer/7.-semester/ma-ala1-1-1

## Structure

This repository contains supplementary course material to the Itslearning page of the course and Wiseflow where the assignments can be found and handed-in. 

- `Lessons code`: includes code solving the examples presented in the lessons. 
- `Hand-ins material`: includes supplementary material to the hand-ins, such as the data files.

## Libraries

The following libraries are used in the code examples and need to be installed in order to run the code:
- `sympy`
- `matplotlib`
- `statsmodels` (only for 11_PCA)
- `sklearn` (only for 11_PCA)

You are welcome to use other libraries, such as `numpy` and `scipy` if you wish to do that. The file `00_Python_Introduction.ipynb` includes examples of comparison between `numpy`/`scipy` and `sympy` for matrix operations.
